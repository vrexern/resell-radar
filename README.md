# Resell Radar

Resell Radar is a personal product comparison and deal-monitoring project for sneakers, streetwear, limited products and events.

The project analyzes authorized retailer and product feeds, compares prices and availability, monitors upcoming releases and highlights potentially interesting offers.

Selected product links and release information may be shared with a Discord community.

All purchases are completed manually. Resell Radar does not use automated checkout, CAPTCHA bypassing, queue circumvention, purchase-limit bypassing or other anti-bot evasion methods.
